# useSpring

`useSpring`은 스프링 애니메이션으로 최신 대상으로 애니메이션될 [motion value](/docs/motion-values/overview)를 생성합니다.

대상은 `.set`을 통해 수동으로 설정하거나, 다른 motion value를 전달하여 자동으로 설정할 수 있습니다.

## 사용법

Motion에서 `useSpring`을 임포트하세요:

```jsx
import { useSpring } from "motion/react";
```

### 직접 제어

`useSpring`은 숫자로 생성될 수 있습니다:

```jsx
const x = useSpring(0);
```

이제 이 motion value가 `set()`을 통해 업데이트될 때마다, 값은 정의된 스프링으로 새로운 대상으로 애니메이션됩니다.

```jsx
x.set(100);
```

[`jump()` 메서드](/docs/motion-values/overview)를 사용하여 스프링 없이 즉시 이 값을 업데이트하는 것도 가능합니다.

```jsx
x.jump(100);
```

### 다른 motion value 추적

다른 motion value의 최신 값을 자동으로 스프링하는 것도 가능합니다:

```jsx
const x = useMotionValue(0);
const y = useSpring(x);
```

### 트랜지션

`spring`의 유형은 일반적인 [스프링 트랜지션 옵션](/docs/animation/transition)으로 정의할 수 있습니다.

```jsx
useSpring(0, { stiffness: 300 });
```
